# Пустошь: Нулевой Круг — обратная связь беты

Публичный репозиторий для бета-фидбека по игре «Пустошь: Нулевой Круг». Сам код игры приватный — здесь только баги, идеи, вопросы.

🎮 **Играть:** https://way-circle-game.vercel.app/#character

---

## Как сообщить

1. Вкладка **[Issues](../../issues)** наверху → **New issue**.
2. Выбери шаблон: 🐛 баг / 💡 идея / ❓ вопрос.
3. Заполни и отправь.

Нужен бесплатный GitHub-аккаунт: https://github.com/signup

---

## Примеры хороших репортов

- **Антилорный текст:** «В квесте Страта на шаге „Обжечь первую посуду“ видно „Серебрушка ×2 · гарант.“ — слово гарант режет глаз.»
- **Карточки нет:** «В подтабе „Действия“ нет карточки „Добыть камень“ — нельзя добывать большой и острый камень.»
- **Не хватает инфо:** «В карточках рецептов еды нет указания, сколько восстанавливает выносливости — невозможно сравнить, что выгоднее крафтить.»
- **Награда не пришла:** «Завершил квест Мирголо, в тосте только „Шаг выполнен!“ без названия предмета.»
- **Квест застрял:** «Квест Страта, шаг „Обжечь первую посуду“ — обжёг 5 горшков, шаг не закрывается.»

Главное: **что произошло** + **что ожидалось** + конкретные имена (квест/NPC/ресурс) + скриншот, если визуальное.
